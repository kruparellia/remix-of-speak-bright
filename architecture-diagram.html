End-to-End Data Flow — Trainer Mode
1
User Starts Drill
Selects topic + difficulty; app requests mic permission
2
Browser APIs Active
AudioContext + SpeechRecognition start; FFT frames stream in real-time
3
FillerDetector Running
Each animation frame analyzed; callbacks fire on filler / silence detection
4
Drill Completes
Stats built (WPM, clarity, filler counts); session record inserted into Supabase
5
AI Analysis
Transcript + topic sent to Edge Function → Gemini 2.5 Flash → structured feedback returned
6
Dashboard Updates
AI scores written back to DB; Dashboard queries sessions and re-renders charts & insights
